\# SNET Chatbot



A chatbot for the SNET Digitization Archive built using Dify.



\## Features

\- Search disaster risk reduction articles

\- Public chatbot interface

\- Custom prompts

\- Docker deployment

\- Temporary public hosting with ngrok



\## Project Structure

\- frontend/: chatbot webpage

\- prompts/: system prompt

\- docker/: deployment files





\## Run Locally

1\. Start Docker Desktop

2\. Run docker compose up -d

3\. Open http://localhost



\## Public Access

Run ngrok http 80 and use the generated public link.
